# Python 接口自动化测试 Demo

这个仓库是我个人练习的接口自动化测试项目。

## 技术栈
- Python
- Pytest（测试框架）
- Requests（API 请求）
- Pandas（数据处理）

## 如何运行
1. 克隆仓库：`git clone https://github.com/你的用户名/仓库名.git`
2. 安装依赖：`pip install -r requirements.txt`
3. 运行测试：`pytest`
